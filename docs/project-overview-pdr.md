# Project Overview - Ken Laam Blog

## Summary

Personal blog and portfolio site for Ken Laam, a Senior iOS Developer. Built with Jekyll and deployed on Cloudflare Pages.

## Project Goals

1. **Personal Brand**: Showcase iOS development expertise and indie apps
2. **Knowledge Sharing**: Blog posts about Swift, iOS development, and lessons learned
3. **App Portfolio**: Showcase published App Store apps with links to privacy policies

## Target Audience

- iOS developers seeking tutorials and tips
- Potential employers/clients reviewing portfolio
- App users looking for support/privacy information

## Tech Stack

| Component | Technology |
|-----------|------------|
| Static Site Generator | Jekyll 4.3 |
| Styling | SCSS (custom dark theme) |
| Fonts | Inter, JetBrains Mono (Google Fonts) |
| Hosting | Cloudflare Pages |
| Ruby Version | 3.2.2 |

## Key Features

- **Blog**: Posts with syntax highlighting, categories, RSS feed
- **Apps Page**: Portfolio of published iOS apps
- **About Page**: Personal introduction
- **Dark Theme**: GitHub-inspired developer aesthetic
- **SEO**: jekyll-seo-tag integration
- **Responsive**: Mobile-first design
